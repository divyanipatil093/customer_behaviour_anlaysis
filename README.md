Data Analyst Project
🔹 Overview
This project demonstrates a complete end-to-end data analysis workflow, starting from raw data extraction to delivering actionable insights through dashboards and presentations.
The objective of this project is to analyze data, identify patterns, and support data-driven decision-making using industry-standard tools like Python, SQL, and Power BI.

🔹 Dataset
The dataset used in this project is sourced from PostgreSQL database and contains data related to:
Sales transactions / Customer data / Property management / etc.
Key Features of Dataset:
Customer ID
Product / Service Details
Transaction Date
Revenue / Amount
Location
Category

These attributes are used to generate insights and build dashboards.
🔹 Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
→ Data loading, cleaning, and analysis
PostgreSQL
→ Data extraction using SQL queries
Power BI
→ Dashboard creation and data visualization
Gamma
→ Presentation and storytelling
🔹 Project Workflow
1️⃣ Data Loading
Imported dataset from PostgreSQL into Python
Used libraries like pandas for structured data handling
2️⃣ Exploratory Data Analysis (EDA)
Checked dataset shape, data types
Analyzed distributions and trends
Identified correlations between variables

✔ Example:
Sales trends over time
Top-performing categories
Customer behavior patterns
3️⃣ Data Cleaning
Handled missing values
Removed duplicates
Treated outliers
Standardized data formats
4️⃣ SQL Queries
Performed analysis using PostgreSQL:
✔ Example Queries:
Total revenue by category
Monthly sales trend
Top customers
Region-wise performance
5️⃣ Dashboard Building (Power BI)
Created an interactive dashboard with:
KPI Cards (Revenue, Profit, Count)
Bar Charts (Category-wise analysis)
Line Charts (Trend analysis)
Filters / Slicers for interactivity
6️⃣ Report Creation
Combined insights from Python & SQL
Structured findings into a readable format
7️⃣ Presentation (Gamma)
Created a professional PowerPoint
Included:
Problem statement
Key insights
Visual dashboards
Final conclusions
